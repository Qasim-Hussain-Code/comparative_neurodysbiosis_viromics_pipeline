# Raw Sequence Data

This directory contains the raw paired end FASTQ sequencing reads downloaded from the European Nucleotide Archive (ENA).

### Contained Files
* `ERR3144321_1.fastq.gz` and `ERR3144321_2.fastq.gz` (Neurodysbiosis Cohort - ASD donor mouse)
* `ERR3144319_1.fastq.gz` and `ERR3144319_2.fastq.gz` (Neurotypical Control - TD donor mouse)

These large data files are excluded from version control.
