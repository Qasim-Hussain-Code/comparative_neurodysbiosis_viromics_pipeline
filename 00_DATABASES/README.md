# Pipeline Reference Databases

This directory contains the local databases required by the annotation and grading software.

### Contained Databases
* `genomad_db` (Database for geNomad taxonomy and marker classification)
* `checkv-db-v1.5` (Database for CheckV completeness and quality estimation)
* `PhrogDB_v14` (PHROG functional database)
* `Pfam_A_DB` (Pfam protein families database)

These database structures are excluded from version control.
